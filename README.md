 Nestlé HR Assistant using Generative AI

## Overview
This project focuses on developing an AI-powered HR Assistant capable of answering employee queries based on Nestlé's Human Resources Policy document. The application uses OpenAI's GPT model to process user questions and generate relevant responses from the HR policy document through a user-friendly Gradio interface.

## Project Highlights
* PDF document processing
* Text extraction from HR policy documents
* OpenAI GPT integration
* Conversational question-answering system
* Prompt engineering for HR-related queries
* Interactive Gradio user interface

## Technologies Used
* Python
* OpenAI API
* Gradio
* PyPDF
* Python Dotenv

## Results
* Developed an AI-powered chatbot capable of answering questions related to Nestlé HR policies.
* Successfully integrated PDF document processing with a Large Language Model.
* Built an interactive interface for efficient information retrieval.
* Improved accessibility of HR policy information through natural language interactions.

## Skills Demonstrated
* Generative AI Application Development
* Prompt Engineering
* Large Language Models (LLMs)
* Document Processing
* API Integration
* Conversational AI
* User Interface Development

## Conclusion

This project demonstrates the practical application of Generative AI and Large Language Models in enterprise environments. By combining OpenAI's GPT capabilities with document processing and an interactive interface, the solution enables users to efficiently retrieve information from HR policy documents through natural language conversations.
